For loops

Loops are used to execute a set of statements repeatedly until a particular condition is satisfied. In Java we have three types of basic loops: for, while and do-while. In this tutorial we will learn how to use “for loop” in Java.

Its syntax is given below:

for(initialization; condition ;update)
{
 //code block to be executed
}

Flow diagram

Example

class ForLoop Example {
public static void main(String args[]){
              for(int i=0; i<10; i++){
        System.out.println("The value of i is: "+i);
      }
    }
  }

Its output is:

The value of i is: 0
The value of i is: 1
The value of i is: 2
The value of i is: 3
The value of i is: 4
The value of i is: 5
The value of i is: 6
The value of i is: 7
The value of i is: 8
The value of i is: 9

Enhanced For loop

Enhanced for loop is useful when you want to iterate Array/Collections, it is easy to write and understand.

Example 
class ForLoop ExampleEnhance {
public static void main(String args[]){
           int arr[]={2,11,45,9};
           for (int num : arr) {
              System.out.println(num);
      }
    }
  }

Its output is:

2
11
45
9
